## Usage Instructions:
1. Copy the prompt template above
2. Fill in your specific project details in the brackets
3. Paste the complete prompt into your AI assistant
4. Copy the generated README content into your GitHub repository
5. Customize any sections that need project-specific adjustments
6. Add actual links to your files, datasets, and visualizations

## Pro Tips:
- Keep the README updated as your project evolves
- Add screenshots of your key visualizations
- Include a demo or example output
- Make sure all links work before publishing
